Installation Guide

From Stock ROM

1. Unlock bootloader
2. Flash ROM via Odin (use CSC, not HOME_CSC)
3. Format data in TWRP
4. Reboot & enjoy!

From Custom ROM

1. Flash ROM via Odin
2. Reboot device
3. Format data if issues arise
4. Done!
