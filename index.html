Installation Guide

Easy steps to get OneAI running on your device

From Stock ROM or Other Custom ROM

1. Ensure you have an unlocked bootloader
2. Flash the ROM with Odin using CSC file (not HOME_CSC)
3. Boot into TWRP and perform Format Data
4. Reboot and enjoy OneAI!

From Older OneAI Releases

1. Flash the ROM with Odin using HOME_CSC file
2. Reboot your device and enjoy
3. If issues occur, boot into TWRP and Format Data
4. Reboot and enjoy OneAI!
